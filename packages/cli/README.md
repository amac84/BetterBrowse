# @amac84/betterbrowse

CLI-native web app auditor for React and Next.js projects.

## Install

```bash
npm install --save-dev @amac84/betterbrowse
```

## Usage

```bash
npx betterbrowse init
npx betterbrowse doctor
npx betterbrowse audit
npx betterbrowse fix
```

Your app should already be running locally (default `http://localhost:3000`) before auditing.
